# Verax - Subgraph

This subgraph aims to index the data generated by the smart contracts of Verax.

## How to use it?

Our subgraph is deployable on all the networks where Verax is deployed.  

- `hashkey-sepolia`

### 1. Generate code

```bash
yarn codegen:v1
```

### 2. Build the subgraph

```bash
yarn build:v1
```

### 3. Deploy the subgraph

```bash
yarn deploy:v1
```
